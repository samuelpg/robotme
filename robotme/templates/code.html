Variables

Las variables son espacios reservados en la memoria que, como su nombre indica, pueden cambiar de contenido a lo largo de la ejecución de un programa.

Los nombres de tus variables deben tener un nombre descriptivo, y debe seguir las siguientes normas:

  • No debe tener espacios.
  • No deben tener numeros.
  • No deben tener mayusculas.

una buena practica para nombrar variables es de separarlas con "_". brazo_zombie_derecho se lee mejor que brazozombiederecho.

Siempre deben tener un valor inicial:

#Pueden ser cadenas o strings
saludo = 'Hola Mundo'
#o numeros
edad = 14

Darle un valor inicial a una variable se llama 'inicializar', y debemos hacerlo todo el tiempo.

Movimiento

Los elementos capaces de causar moviemiento son los servos y los motores DC.

Servo

Para controlar un servo tienes la instruccion girar, debes usarla de esta manera:

#girar [variable] [posicion] grado
#Ejemplo: esta linea gira 'brazo' al grado 0
girar brazo 0 grados

Puedes girar un servo a las posiciones 0, 90 y 180 grados.

Motor

Para controlar un motor tienes las instrucciones mover y parar, debes usarla de esta manera:

#Para mover un motor
mover rueda
#Para parar un motor
parar rueda

Salida

Los elementos que pueden ser utilizados en los pines de salida son los leds y los buzzers.

LEDS

Los LEDS son diodos emisores de luz, los cuales tienen la función de convertir la corriente eléctrica de bajo voltaje en luz. Para controlar un LED tienes las siguientes intrucciones: encender, debes usarla de esta manera:

#encender
#Ejemplo: esta linea enciende el LED
encender led

apagar, debes usarla de esta manera:

#apagar
#Ejemplo: esta linea apaga el LED
apagar led

Buzzer

Un buzzer es un elemento cuya función es transformar la energía eléctrica en acústica, de manera que se emita un sonido. Para controlar un buzzer tienes las siguientes instrucciones: encender y debes usarla de esta manera:

#Para encender el buzzer
#Ejemplo: esta linea enciende el buzzer
encender buzzer

apagar, debes usarla de esta manera:

#Para apagar el buzzer
#Ejemplo: esta linea apaga el buzzer
apagar buzzer

Otros

Comentario

Los comentarios son utiles para explicar tu codigo ya que son ignorados al momento de correr el codigo, comienzan con el simbolo #

#Ejemplo:
#Este codigo no se ejecutara nunca
#encender led

Decir

La consola del sistema es un terminal que tiene atributos especiales y se utiliza para determinados fines. En este caso, el fin sería el de verificar como esta ejecutandose el programa previamente realizado. Para poder utilizar la consola de esta manera tienes las siguientes instrucciones: decir [mensaje], debes usarla de esta manera:

#Ejemplo:
#Puedes usar decir con numeros
decir 24
#o con variables
decir mensaje
#Tambien con oraciones entre comillas
decir 'Hola mundo!'
#Y combinando con el simbolo +
decir 'Saludos' + nombre

Esperar

La finalidad de esta función es la de detenr la ejecución del programa por una cantidad de segundos determinada. Para poder detener el programa tienes la siguiente instruccion: esperar [numero] segundos, debes usarla de esta manera:

#Ejemplo:
esperar 24 segundos
